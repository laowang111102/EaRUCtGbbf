# 选课管理系统 java1242

## 项目概述

选课管理系统是一个基于前后端分离模式的教育管理系统，旨在提供便捷的课程选择、管理功能。系统主要包括管理员、教师、学生三种角色，以实现不同的业务需求。

## 技术栈

- **服务端技术**：Springboot + Mybatis-Plus
- **前端技术**：HTML + CSS + Vue + Element-UI + Axios
- **运行环境**：JDK 8 + MySQL 5.7
- **开发工具**：IntelliJ IDEA + Maven + Node.js

## 功能模块

### 管理员

- 院系管理
- 专业管理
- 班级管理
- 课程管理
- 选课管理
- 学生管理
- 教师管理
- 管理员管理

### 教师

- 授课查询
- 教师课表
- 成绩录入

### 学生

- 选课
- 已选课程管理
- 课表查询
- 成绩查询
- 信息维护

## 系统角色

系统定义了以下三种角色：

- 管理员：负责系统维护、用户管理、课程管理等任务。
- 教师：主要负责课程教学、成绩录入等教学活动。
- 学生：进行课程选择、成绩查询等学习活动。

## 运行环境

- JDK 8 或更高版本
- MySQL 5.7 或更高版本
- IntelliJ IDEA
- Maven
- Node.js

## 目录结构

```

|-- /backend 服务端代码目录
|   |-- /src
|       |-- /main
|           |-- /java
|           |-- /resources
|-- /frontend 前端代码目录

```

## 部署步骤

1. 准备运行环境：安装 JDK 8、MySQL 5.7、Node.js。
2. 编译后端代码：使用 IntelliJ IDEA 导入后端代码，运行 Maven 命令以编译项目。
3. 部署后端服务：执行后端程序。
4. 编译前端代码：进入前端项目目录，运行 npm install 安装依赖，然后运行 npm run build 生成静态文件。
5. 部署前端文件：将生成的静态文件放置到 Web 服务器目录。
6. 访问系统：通过浏览器访问前端页面。
